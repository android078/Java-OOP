При написании программы были использованы отдельные родственные классы, где каждый обрабатывает одно арифметическое действие.
Их три - это сложение, умножение, деление.
Точка запуска - код Calculator. В нем идет инициализация классов арифм. действий (через модуль-класс Init).
Там же создается Map с символами арифметических  действий и их названиями, запускается главное меню (UserDialog).
